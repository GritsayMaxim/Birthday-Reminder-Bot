# Birthday-Reminder-Bot
Telegram-бот для напоминаний о днях рождения с генерацией персонализированных поздравлений   Умный Telegram-бот, который помогает не забывать дни рождения друзей и коллег. Автоматические напоминания, готовые поздравления с учетом возраста, настраиваемое время уведомлений. Все данные хранятся локально.
